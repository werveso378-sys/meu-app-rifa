# 🧱 MODO SERVIDOR ISOLADO (BACKEND SEGURO)

**ATENÇÃO IA:** O usuário pediu para trabalhar no Backend.

## A RESTRIÇÃO:
Você deve trancar o seu escopo mental às pastas de Servidor/Banco de Dados (ex: `api/`, `_services/`, funções de Firebase, e Webhooks).

1. É **PROIBIDO** tocar em componentes visuais `.jsx` ou `.css` do frontend durante esta sessão de backend.
2. O fluxo de dados da API para o Frontend deve ser garantido via resposta JSON padronizada, mas o comportamento de renderização não deve ser modificado neste contexto.
