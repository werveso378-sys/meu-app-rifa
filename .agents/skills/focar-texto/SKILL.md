# 📝 MODO REDATOR (TEXTOS ISOLADOS)

**ATENÇÃO IA:** O usuário pediu apenas para alterar textos, frases, títulos ou alertas.

## A RESTRIÇÃO:
O seu único e exclusivo objetivo é alterar strings contendo português.

1. Você deve modificar APENAS strings dentro de aspas. 
2. Você está TERMINANTEMENTE PROIBIDA de alterar o nome de variáveis lógicas, lógicas if/else, nomes de funções que disparam ações no código, ou reestruturar as tags de HTML. Altere APENAS o conteúdo humano lido no meio das tags ou aspas de alert.
