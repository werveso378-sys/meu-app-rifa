# 🪲 MODO INVESTIGAÇÃO (DEBUG SEGURO)

**ATENÇÃO IA:** O usuário reportou um bug e precisa de uma investigação controlada.

## A RESTRIÇÃO:
Você está **PROIBIDA** de apagar ou reescrever a lógica do aplicativo na tentativa desesperada de consertar.

1. **Atue como um investigador forense.** Limite-se a adicionar estruturas inofensivas como `console.log()` ou blocos de `try/catch` para capturar o fluxo de variáveis.
2. Descubra a CAUSA RAIZ através dos logs e dados em tempo real.
3. Não presuma soluções. Primeiro documente a causa e sugira a solução ao usuário de forma clara. Só aplique o código corretivo estrutural após ter 100% de certeza do que quebrou.
